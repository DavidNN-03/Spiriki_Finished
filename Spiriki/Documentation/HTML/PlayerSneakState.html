public class PlayerSneakState


Namespace: Spiriki.Control

Inherits from: PlayerBaseState


State for when the player is sneaking.



private readonly intSneakSpeedHashHash for the parameter SneakSpeed.
private readonly intSneakBlendTreeHashHash for the sneaking blend-tree.
private const floatCrossFadeDurationThe amount of seconds to crossfade to the sneak animation.
private const floatAnimatorDampTimeAnimation damp time. Defines how long it takes to change to change the FreeLookSpeed parameter. A high value means that the parameter will change more slowly.


public override voidEnter()Called when entering state. Subscribes to Jump-, and Sneakevent. Also, crossfades to sneaking blend-tree.
public override voidTick(float deltaTime)Called every FixedUpdate. Applies gravity and moves the player. Also, if the player is not grounded, switch to PlayerFallingState.
public override voidExit()Called when exiting the state. Unsubscribes from Jump- and SneakEvent.
private voidHandleSneak(bool shouldSneak)Sets if the player should sneak. If notm switch to PlayerFreeLookState.
private voidHandleJump()If the player is grounded, switch to PlayerEnterJumpState.
private Vector3CalculateMovement()Calculate which direction the player should move in.
